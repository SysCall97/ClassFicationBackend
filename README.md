# ClassFicationBackend

## Installation
```bash
 Step-1: 
 $ git clone https://github.com/SysCall97/ClassFicationBackend.git

 Step-2:
 Setup environment variables in .env file in the root directory

 Step-3:
 $ npm install

 Step-4:
 $ npm run dev
```

## Environment variables
```bash
 - URI
 - JWT_SECRET
 - JWT_EXPIRE_MINUTE
```

## Technology used
 - Node JS
 - Express JS
 - TypeScript
 - MongoDB
 - Mongoose
